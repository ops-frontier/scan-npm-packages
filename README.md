# scan-npm-packages
Shai-Hulud に汚染されたパッケージを含むレジストリを探すツールを実装するリポジトリです。

## 仕様

- プログラムは JavaScript で記述して、node.js で実行する
- パッケージマネージャには　pnpm を使用する
- 環境変数 GITHUB_ORGANIZATION に指定された Github 組織に属するすべてのリポジトリを巡回する
- 環境変数 GITHUB_TOKEN を使用して Github にアクセスする
- リポジトリ内にある pakcage-lock.json, yarn.lock, pnpm-lock.yaml が [package-list.md](package-list.md)に記載しているパッケージを参照していないかをチェックする
- package-list.md に記載のバージョンと照合する必要はなく、パッケージを参照しているかどうかだけをチェックする
- パッケージが見つかった場合は、その参照しているバージョンとともにリポジトリ名、ファイルパスを印字する
- 5個を上限として並列に実行する

## Excel への出力

スキャンした内容全体を shai-hulud-scan.xlsx という名前の Excel ファイルに出力する。

- ファイルが存在する場合は上書きする
- シートは一つだけで、最初にヘッダ行を出力する
- 列の内容は A: リポジトリ, B: ファイルパス, C: パッケージ名, D: パッケージバージョン、 E: 対象パッケージ数、F: 汚染パッケージ数 とする
- パッケージの参照が見つかるごとに、1行を出力する
- 汚染されているか否かには関係なく見つかったパッケージを全て出力する
- パッケージ名が  [package-list.md](package-list.md) に含まれている場合は E: 対象パッケージ数 を 1 とし、含まれていない場合は 0 とする
- パッケージのバージョンが  [package-list.md](package-list.md) の汚染バージョン（カンマ区切り）のどれかと一致している場合は E: 汚染パッケージ数 を 1 とし、一致しない場合は 0 とする
